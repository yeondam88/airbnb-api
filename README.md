# Airbnb API

REST & GraphQL API of the Airbnb Clone using Django REST Framework and Graphene GraphQL

### API Actions

- [ ] JWT Authentication
- [ ] Login (JWT)
- [ ] Create Account
- [ ] See Profile
- [ ] Edit Profile
- [ ] Add/Room From Favourites
- [x] List Rooms
- [x] See Room
- [x] Create Room
- [x] Edit Room
- [x] Delete Room
- [ ] Search Rooms

## API Endpoint

/api/v1/users/token/ - Sign in
/api/v1/users/ - Sign up
